# Developer Documentation Archive

This directory stores dated development notes, local experiment records, and
maintenance notes that are useful for audits but should not be treated as the
stable user-facing documentation.

Path placeholders used in archived commands:

- `<PROJECT_ROOT>`: local Sparse-vLLM checkout.
- `<MODEL_ROOT>`: local model directory.
- `<CHECKPOINT_ROOT>`: local DeltaKV compressor checkpoint directory.
- `<DATA_ROOT>`: local benchmark dataset directory.
- `<OUTPUT_ROOT>`: local benchmark/log output directory.
- `<SVLLM_PYTHON>`: Python executable for the project environment.
- `<REVIEW_CONDA_ENV>`: review or validation conda environment path.
